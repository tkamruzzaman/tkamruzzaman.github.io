Kitchen Chaos

Co-op multiplayer game using NetCode for Gameobjects. Cook fast food orders by following different recipes and deliver them to the customer before the time runs out, either in single-player or in co-op multiplayer mode.

Singleplayer work: Implemented Character Controller, Collisions and Interactions, Objects pick up and drop, Animation, C# Events and Interfaces, Shader Graph, Post Processing, New Input System, Key Rebinding, and Controller Input, UI, Menu Navigation and Loading, Music and Sound Effects, State Machines, Scriptable Objects.

Multiplayer work: Client vs Server Auth, Netcode and Basic Setup, Sync Player Movement, Animation and collisions, Pick up and drop Objects, Sync Delivery Manager, Plates Counter, Trash Counter, Cutting Counter, Stove Counter, and Plate Add Ingredients, Sync Game States and Pause, Handle Disconnects, Connection Scene Flow, Character Selection Scene, Implement Lobby, Implement Relay, Multiplayer Debugging Tools, Add Singleplayer Option, Add support for Gamepad
